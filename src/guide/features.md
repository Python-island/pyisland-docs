# 功能介绍

::: info
本文档详细介绍 Pyisland 的各项功能。不同分支的功能可能略有差异，以各分支实际实现为准。
:::

## 界面设计

### 灵动岛外观

- **胶囊形状**: 采用现代胶囊形状设计，模拟 iOS 灵动岛
- **实时时间**: 初始状态显示当前系统时间
- **可拖动**: 支持鼠标拖动调整灵动岛位置
- **自动吸附**: 支持吸附至屏幕上边缘、左侧、右侧

### 展开/收起

- **点击展开**: 点击灵动岛展开显示控制面板
- **自动收缩**: 展开后失去焦点自动收缩
- **平滑动画**: 展开/收起过程有流畅的过渡动画

## 系统控制

::: tip 亮度调节
- 滑动条调节系统屏幕亮度
- 支持防抖机制，避免频繁调节
- 实时显示当前亮度百分比
:::

::: tip 音量控制
- 滑动条调节系统音量
- 支持防抖机制
- 实时显示当前音量级别
:::

## 系统监控

### 状态显示

实时监控并显示以下系统状态：

| 状态 | 说明 |
|------|------|
| WiFi | 当前 WiFi 连接状态和信号强度 |
| 蓝牙 | 已连接的蓝牙设备 |
| 电池 | 电池电量和充电状态 |

### 通知提醒

- WiFi 连接/断开时自动提醒
- 蓝牙设备连接/断开时自动提醒

## 剪贴板功能

### URL 检测

- 自动监控剪贴板内容
- 检测到 URL 时显示提示
- 提供快捷按钮打开链接

### 全局热键

部分分支支持全局快捷键（如 `Alt+O`）快速打开复制的链接。

## 媒体控制 (部分分支)

::: info pyisland-wanku 特有
部分高级分支（如 `pyisland-wanku`）支持：
- 自动识别正在播放的音乐
- 显示歌曲名称和艺术家
- 播放/暂停、上一曲、下一曲控制
- 音乐可视化动画
:::

## 录屏功能 (部分分支)

::: warning pyisland-wanku 特有
`pyisland-wanku` 分支提供：
- 一键录屏
- 自动保存为 MP4 格式
- 实时录制计时器
- 录制状态呼吸提示
:::

## 系统托盘

### 托盘菜单

- **设置**: 打开设置窗口
- **退出**: 退出应用程序

### 托盘行为

- 最小化时隐藏到系统托盘
- 双击托盘图标打开设置
- 右键显示菜单

## 设置选项

::: tip 可用设置
根据不同分支，可能包含以下设置：
- 启动时运行
- 开机自启
- 主题选择（浅色/深色）
- 歌词显示模式
- 快捷键配置
- 剪贴板监控开关
:::
